## 💡 設計理念：用「打包行李」直覺學習 Flexbox

對於網頁設計的初學者（程式小白）來說，CSS 的排版屬性（如 `justify-content`、`align-items`、`flex-wrap` 等）往往像是一堆抽象的魔法咒語。看著代碼，卻很難在腦袋裡想像出它們在瀏覽器中實際移動的模樣。

為了解決這個痛點，**Flexbox Luggage** 誕生了。

我們將抽象的「網頁元件排版」轉化為大家都有經驗的日常情境——**「收拾行李箱」**。

### 📦 核心隱喻 (Metaphor)
* **行李箱 (Suitcase)** = `flex-container` (彈性容器)
* **衣服、相機、洗漱包 (Luggage Items)** = `flex-items` (彈性元件)

當你要把護照、相機、衣物塞進去時，你需要決定它們要**橫著放還是直著放** (`flex-direction`)、要**靠左靠右還是居中** (`justify-content`)、甚至在行李太多時**自動折行** (`flex-wrap`)。透過這個生活化的比喻，原本枯燥的 CSS 屬性瞬間變得直覺且容易理解。

## 🎨 遊戲特色與視覺設計

為了給玩家帶來最棒的學習體驗，我們在細節上做了許多精雕細細琢：

1. **左右對照，直覺回饋**
   * 左側為「理想收納狀態（目標樣式）」，右側為「你的行李箱（即時預覽）」。
   * 玩家不需要盲猜代碼效果，每次切換下拉選單，行李箱內的物品就會立刻動起來，提供最即時的視覺反饋。
2. **溫暖優雅的視覺調色盤**
   * 捨棄了高飽和度的刺眼配色，採用溫暖、有生活質感的色調：燕麥色（#FFF8F0）的行李箱內襯、莫蘭迪色系的行李元件、以及活潑但不失質感的暖橘色調。
3. **由淺入深的 7 大關卡**
   * **Level 1 ~ 3：基礎對齊**（對齊相機、護照，學習 `justify-content` 與 `align-items`）。
   * **Level 4 ~ 6：單一元件調整**（學習 `flex-grow` 與 `align-self`，體驗物品如何「擠滿」或「單獨靠邊」）。
   * **Level 7：環遊世界大挑戰**（綜合運用 `display: flex`、`flex-wrap` 與對齊，整理大量行李）。
